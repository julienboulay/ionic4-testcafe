# ionic4-testcafe

## Prerequisites

Install ionic :

```
  npm install -g ionic
  npm install -g testcafe
```

## Installation

Install npm dependencies :

```
  npm install
```

## Tests

### Command line
First run the app

```
  ionic serve
```

Then run the tests

```
  npm run e2e
```

### Testcafe studio

Install [testcafe studio](https://www.devexpress.com/products/testcafestudio/)
